資料欄位說明
固定面板資料集的 20 個欄位
欄位說明
timestamp10 分鐘解析度時間戳（Asia/Taipei, UTC+8）
tilt_angle面板傾角 (10° / 15° / 20° / 30°)
azimuth_angle面板方位角 (160° / 180° / 200°)
panel_id面板識別碼（如 Panel_15_180_A 表示 15° 傾角、180° 方位、A 片）
voltage_V / current_A / power_W該時刻面板輸出（V / A / W）
daily_energy_Wh當日累計能量（取每日 max 即為當日總能量）
solar_zenith / solar_azimuth / solar_elevation太陽天頂角 / 方位角 / 仰角（pvlib 計算）
theoretical_poa理論面板入射輻照度 W/m²
airmass / dni / dhi / ghi大氣質量 / 直射 / 漫射 / 全天輻射 (W/m²)
is_tracking是否為追日面板（固定面板為 False）
